---
title: "E-Clair"
---

Personal finance tracking that doesn't suck.

Rocket Money and Monarch are fine if you're okay with slow, bloated, and subscription-priced. Copilot Money *looks* vibe coded and the UI is entirely overwhelming. E-Clair is the version I actually want to use: fast, local-first, and built on Plaid for bank connections.

The core is written in Rust for the data processing layer — transaction categorization, deduplication, and rollup queries need to be instant. The frontend is TypeScript/React. Plaid handles the bank OAuth and transaction sync.

Still in progress — but the goal is a tool that covers budgeting, net worth tracking, and spending analysis without the middleman markup. If finished and released then will be funded by tips - not another subscription.
